# Conquista App 🚀

**Conquista App** é um ecossistema de alta performance desenvolvido para gestão ágil de vendas, comissões e bônus. O sistema foi projetado com uma interface moderna, rápida e focada em resultados.

## ✨ Funcionalidades Principais

- **📊 Dashboard de Performance:** Acompanhamento em tempo real de metas de produtos, assistência e impermeabilização.
- **💰 Gestão de Comissões:** Cálculo automático de comissões base (2.2%), bônus de serviços fixos e bônus acelerador por nível de performance.
- **🚫 Cancelamento Inteligente:** Sistema de cancelamento de pedidos que mantém o histórico mas remove automaticamente os valores dos cálculos financeiros.
- **📱 Relatórios para WhatsApp:** Geração de relatórios formatados para compartilhamento rápido com supervisores e equipes.
- **🌐 Offline-First:** Funciona perfeitamente sem internet, sincronizando dados localmente via `LocalStorage`.
- **🎨 Interface Moderna:** Design otimizado para produtividade, com animações suaves e tipografia marcante.

## 🛠️ Tecnologias Utilizadas

- **React 19+**
- **TypeScript**
- **Tailwind CSS** (Estilização Utilitária)
- **Lucide React** (Ícones)
- **Motion** (Animações)
- **Vite** (Build System)

## 🚀 Como Rodar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/montador2021-netizen/conquista-app-oficial.git
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd conquista-app-oficial
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Abra no navegador:**
   O projeto estará rodando em `http://localhost:3000`.

## 📂 Estrutura de Arquivos

- `src/App.tsx`: Componente principal e lógica de estado.
- `tipos.ts`: Definições de interfaces e enums do TypeScript.
- `components/`: Componentes reutilizáveis (Formulário, Sidebar).
- `constants.tsx`: Constantes e configurações do sistema.
- `src/index.css`: Configurações globais de estilo e Tailwind.
